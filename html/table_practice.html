Programming Languages Description
HTML A markup language that controls the structure of websites, and facilitates linking web pages through HTTP and HTTPS
CSS A cascading set of style rules that key onto specific HTML tags, IDs, or Classes that comes together to control the visual appearance of websites
Ruby A dynamic, object-oriented, general purpose scripting language that is a favorite among new programmers because of it's simple syntax that makes it easy to learn, and because it is in high demand in the the tech industry.
